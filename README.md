## 데이터 분석 기반 영화 콘텐츠 추천 시스템

데이터 분석 과정을 거쳐 도출한 인사이트를 기반으로 기획, 개발하게 된 **사용자 맞춤형 영화 콘텐츠 추천 서비스**입니다. 

- 사용자의 정보를 토대로 개개인이 선호할 만한 맞춤형 영화들을 추천해줍니다.
- 추천받은 영화들 중 선호 영화를 클릭하면 해당 영화와 비슷한 줄거리의 영화까지 추천받을 수 있습니다.
